# Archived Harness Reference

이 폴더는 Work Bench 하네스 설계 배경과 적용 후보를 보관한 archive다. active 작업 규칙이나 필수 gate가 아니라, 왜 이런 구조를 생각했는지 확인하기 위한 자료다.

AI 작업자는 사용자가 명시적으로 이 archive 검토를 요청한 경우를 제외하고, 이 폴더의 문서를 현재 작업 가드, 인용 출처, 구현 근거로 직접 참조하거나 활용하지 않는다.

- [background.md](background.md): 페어 온보딩용 한 장 요약.
- [concept.md](concept.md): Work Bench, Playground, Test Bench를 나누는 핵심 원칙.
- [plan.md](plan.md): 단계별 적용 후보와 run artifact 계약 초안.
- [research/](research/): 외부/기존 사례 조사와 종합 분석.

실제 작업 기준은 루트 가이드와 최신 README를 우선한다. 이 폴더의 내용 중 현재 기준으로 필요한 부분은 규칙, 코드, 테스트, README로 승격한 뒤 사용한다.
