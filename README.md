# whisper-intent-interruption-detection

Whisper 기반 intent/interruption detection 실험과 구현을 위한 팀 사이드 프로젝트다.

## 현재 상태

코드는 아직 들어오지 않았다 — dev baseline(의존성 핀, ruff, pre-commit + secret scan) 준비 단계.
Quickstart(설치·실행·테스트 명령)는 첫 코드 commit과 함께 채워진다.
